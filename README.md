\# Insurance Risk Analytics



\## Project Overview



This project focuses on exploratory data analysis (EDA) for insurance risk and profitability assessment.



\## Objectives



\- Set up a reproducible Python environment

\- Perform exploratory data analysis

\- Assess data quality

\- Identify patterns in risk and profitability



\## Tools Used



\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- Seaborn

\- Scikit-learn

\- Jupyter Notebook
